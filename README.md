## :dolphin:介绍
### 该项目为[酷狗音乐](http://www.kugou.com/)在线听歌的GUI桌面程序。
 - 项目目录:
    - 爬虫文件: Spiders/API_music_kugou.py
    - Tkinter GUI文件: Spiders/GUI_music_kugou.py
    - exe运行程序: exe/GUI_music_kugou.exe (使用pyinstaller库将python文件打包成可执行的exe程序)
